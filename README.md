# pokemon_spread_helper

Helper riusabili per notebook di spread e damage calc Pokemon.

## Uso nei notebook

Tieni i notebook nella root del repository e importa gli helper condivisi:

```python
from pokemon_spread_helper import *
```

La logica comune vive in `pokemon_spread_helper/helpers.py`; i notebook dovrebbero contenere solo setup, input dei team e analisi specifiche.
